# Project 2 – The Server Commander

## Overview

This project demonstrates how to deploy a web server on Amazon EC2 using Ubuntu Linux and Apache.

## Technologies Used

- AWS EC2
- Ubuntu 26.04 LTS
- Apache2
- SSH
- HTML
- Linux Terminal

## Steps Completed

- Created an EC2 Ubuntu instance
- Configured Security Groups
- Connected using SSH from macOS Terminal
- Installed Apache2
- Started and enabled Apache service
- Created a custom HTML webpage
- Successfully hosted the webpage on AWS EC2
